## 1. 常见概念

### 1.1 数据类型

四种基本标量：整型、浮点型、布尔型、浮点类型
复合类型： 元组(tuple) 和 数组(arrya)

```rs

let a = [3;5] == [3,3,3,3,3]

fn t(x: i32) -> i32 {
    x + 1
    // x + 1; 加分号后，会把表达式变为语句，从而没有返回值导致: () 不匹配 i32
}

```
三元表达式： if condition {x} else {y} x和y的类型必须相同

条件判断： if / else if
循环使用：loop / while / for
